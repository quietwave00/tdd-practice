## 요구사항
1. 숫자를 입력받는다.
   - [ ] 숫자가 아닌 경우 재입력을 받는다.
   - [ ] 공백이 포함된 경우 제거 후 입력 처리한다.
2. 연산 기호를 입력받는다.
   - [ ] 사칙 연산에 해당하는 연산 기호가 아닌 경우 재입력을 받는다.
   - [ ] 공백이 포함된 경우 제거 후 입력 처리한다.
   - [ ]연산 기호가 '='인 경우, 현재까지의 결과를 보여준다.
3. 결과를 보여준 후, 프로그램의 종료 여부를 묻는다.
   - [ ] 선택지(예, 아니오)에 해당되는 선택이 아닌 경우 재입력을 받는다.
   - [ ] '예'인 경우 프로그램을 종료한다.
   - [ ] '아니오'인 경우 계산 연산을 이어간다.
